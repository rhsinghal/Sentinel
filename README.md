# Sentinel
Sentinel Prototype

This prototype uses real anonymized smartwatch health data from the Smartwatch Health Data (Uncleaned) dataset on Kaggle, demonstrating realistic patterns in sleep, activity, and heart rate metrics. https://www.kaggle.com/datasets/mohammedarfathr/smartwatch-health-data-uncleaned?resource=download

This is an early-stage prototype & conversational aid for needfinding research. It demonstrates:

How passive monitoring + active check-ins work together
Context-aware prompts (e.g., "Your heart rate spiked on exam day - how did it go?")
Session preparation features for student counseling services
